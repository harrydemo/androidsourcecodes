QuiteSleep (v2.0)

Contacts help

Help about general functionality of the contacts tab options.

Tasks

  • Set contact as blocked

    • You can select any contact from your contact list, that previously has been synchronized from the contact list of the phone, as blocked contact. Blocked contact is a type of contact blocked by the application when arrive an incoming call.

    You can add contacts as blocked ones by one to one way or adding all contacts as blocked in one step. When you choose a contact, you can see its phone numbers and its emails addresses, you can choose 0 or n (being n the number of phone and emails) options to give to QuiteSleep the faculty send messages as feedback by this two ways.

    Remember, no matter if you don't choose none phone number or mail, the contact will be blocked when you add to the blocked list, this two features are only to send messages to the user when realize a call to you.

    If you choose the phone1, but your contact use its phone2, the incoming call will be blocked, but the message don't be sent (this runs if all other options are established correctly).

  • Remove or edit

    • This function allows to you to remove or edit blocked contacts. You can edit phone numbers or emails that QuiteSleep uses to send messages to user as feedback.

    Also you can remove contacts from the blocked list, by one to one or in one step all together.

  • Synchronization

    • This option re-synchronize the internal database of QuiteSleep which is synchronized with the internal database from the operating system the first time that the application runs.

    Put attention in if you use this option, the previous internal contact list and the previous configuration about who is blocked and who is not, and the log call will be deleted.
