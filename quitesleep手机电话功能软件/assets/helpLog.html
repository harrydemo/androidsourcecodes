QuiteSleep (v2.0)

Log help

Help about general functionality of the Log tab options.

Tasks

  • General description:

    • This tab contains the log activity regarding QuiteSleep. In this window you can will see all information about when QuiteSleep has blocked any incoming call showing the following information:

    1. Name (or unknwon if is an unknown caller)
    2. Phone number
    3. Date
    4. Hour
    5. Number of sent sms
    6. Number of sent email

  • Refresh:

    • With this option menu you can refresh this screen with the updated information about the QuiteSleep activity.

    You must refresh this occasionally because QuiteSleep only refresh it automatically when you switch on your mobile phone and run QuiteSleep for first time.

  • Remove:

    • This option menu is used when you want remove data actually showed in the screen. This action not only remove the showed data but delete all information about QuiteSleep activity in the internal ddbb, so once you'll do it, you don't have any information about previous information.
