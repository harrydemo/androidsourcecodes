QuiteSleep (v2.0)

Schedule help

Help about general functionality of the schedule tab options.

Tasks

  • Setting start time

    • We must set start time, this time is when will begin the range where QuiteSleep block the incoming call.

  • Setting end time

    • We must set end time, this time is when will finish the range where QuiteSleep block the incoming call.

  • Setting days of week

    • We must set days along the week where QuiteSleep will can block the incoming call in conjunction with the schedule.

  • Notes

    • You must to configure completely schedule before to start QuiteSleep service, completely are all, start and en time and days of week, if you switch on QuiteSleep to block incoming calls but you don't done previously the configuration, QuiteSleep won't block nothing.
