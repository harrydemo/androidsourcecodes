QuiteSleep (v2.0)

Settings help

Help about general functionality of settings tab options.

Tasks

  • Mail configuration

    • With this option you can use your Gmail account to send email to users which previously have defined in the contact configuration the possibility to send email when QuiteSleep block the incoming call.

    In this configuration you must provide your Gmail account, password, and a subject and body for the email that will be send.

    QuiteSleep doesn't use this data for anything else to send email.

    You can switch on/off this utility when you want, but before switch on this feature for first time you must to save your configuration.

    Put attention that maybe use of this feature can take additional costs to your mobile bill, beware with this.

  • SMS configuration

    • With this option you can use sms to send messages to users which previously have defined in the contact configuration the possibility to send sms when QuiteSleep block the incoming call.

    QuiteSleep doesn't use this data for anything else to send sms.

    You can switch on/off this utility when you want, but before switch on this feature for first time you must to save your configuration.

    Put attention that maybe use of this feature can take additional costs to your mobile bill, be aware with this.

  • Block configuration (new in QuiteSleep 2.0!!)

    • This section is one of the most important. Here you can choose your best block type in one second, you can choose between:

    1. Block all incoming calls
    2. Block only blocked contacts
    3. Block unknown calls
    4. Block unknown and blocked contacts

    As you see with this options you can block any kind of call that you want.

  • Block type (new in QuiteSleep 2.0!!)

    • Here you can choose between block the incoming call by two different ways:

    1. Silent the incoming call:
      2. When QuiteSleep detect any incoming call and must block it, put the mobile phone is silent mode during the call, when the call is finish, QuiteSleep put the mobile phone into normal mode.

      Be aware that if you put your mobile phone into silent, when you receive a call and QuiteSleep blocking it, when finish it, QuiteSleep put the mobile phone into normal mode, so your previous mode, silent, will be changed.

    2. Block the incoming call:
      3. By this way, when arrives an incoming call and QuiteSleep must block it, the app do the hang up action, so the incoming call will be stop.

      This action is "more hard" by the way the user will be received a cut in its call.

      Be aware that this new feature is developed without use of Android SDK, so could be running bad in some mobile phones or Android versions. This feature have been proved in Nexus One with Android 2.0 to 2.2 and runs very well.

  • Start/stop QuiteSleep service With this button, you will can switch on/off QuiteSleep service, so when you want that QuiteSleep must be run, switch on this button (a notification will appear on the status bar) or switch off QuiteSleep service (then the notification on status bar will be disappear).
