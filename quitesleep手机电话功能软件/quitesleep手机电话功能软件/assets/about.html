QuiteSleep (v2.0)

by Cesar Valiente Gordo

Silence or block incoming calls from several sources as contacts blocked, unknown, all, or mix between them in a predefined hour and day interval. Provides feedback to the caller by email or sms. QuiteSleep is only for versions 2.0 and above.

Information

Please visit the QuiteSleep project in Google Code page for more information.

Recent release history

  • 2.0.1 Translation to Chinese and traditional Chinese. Courtesy of bbs.goapk.com
  • 2.0 More options to specify what do you want block. Before only contacts previously defined, now block all, block defined contacts, block unknown, and block unknown and defined contacts.
  • 2.0 Now users have the option to specify the way to block an incoming call, by silence (the previous only option) and automatically hang up (new!).
  • 2.0 SMS send fixed, now it works very good.

See changes on Google Code for the full changelog.
